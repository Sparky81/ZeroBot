ZeroBot
===

	Coding for the project is assisted by Cooper and miniCruzer on ZeroNET.

*	ZeroNET IRC Network Information:
 	- Host: irc.zeronet.us
 	- Port: 6667
 	- SSL:  6697

*	Once installed use this to start the bot:
	- $ ./core.pl &
